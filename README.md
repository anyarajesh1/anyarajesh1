🚀 Featured Projects

⚡ Vigil AI

An AI-native cybersecurity monitoring platform for real-time threat detection and automated incident response.


Detection Engine: Trained ML models on thousands of attack patterns to identify anomalies, suspicious behavioral sequences, and indicators of compromise (IoCs) across live log streams.
Architecture: FastAPI backend ingests logs from any source — web servers, firewalls, cloud platforms — and routes events through a Grok parsing pipeline backed by Elasticsearch.
Intelligence Layer: Every detected event is automatically mapped to the MITRE ATT&CK framework, giving operators immediate context on attacker tactics, techniques, and procedures.
Response Automation: When threat confidence crosses configurable thresholds, Vigil triggers playbooks autonomously — isolating hosts, revoking credentials, and generating audit-ready incident reports.
Dashboard: React frontend with live threat maps, risk scoring, alert prioritization, and incident timelines built for speed and clarity.


🏥 MedInsight AI

An intelligent medical diagnostics platform that turns symptom inputs into ranked differential diagnoses with clinical context.


Diagnostic Engine: Multi-label classification model trained across 50+ disease categories and 200+ clinical symptoms, returning probability-ranked differentials with confidence scores.
NLP Pipeline: BERT-based model processes unstructured clinical text, extracting relevant entities and mapping them to structured diagnostic features.
Research Integration: Live integration with PubMed and Semantic Scholar surfaces relevant papers, clinical trials, and treatment guidelines for any queried condition in real time.
Report Generation: Automatically produces structured clinical summaries, SOAP notes, and patient-friendly plain-language explanations from raw diagnostic inputs using GPT-4.
Privacy-First: HIPAA-aligned architecture with local inference options, data anonymization pipelines, and full audit logging.
